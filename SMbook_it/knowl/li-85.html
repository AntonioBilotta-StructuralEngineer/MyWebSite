Equazioni indefinite di equilibrio sul dominio
\begin{equation} \text{div}\tens{\sigma} = \vec{0}\quad\;\text{in}\;\body_0\,.\tag{3.1.2} \end{equation}
Ovvero, per componenti, le seguenti tre equazioni scalari
\begin{equation} \begin{array}{l} \frac{\partial\sigma_{11}}{\partial X_1} + \frac{\partial\sigma_{12}}{\partial X_2} + \frac{\partial\sigma_{13}}{\partial X_3} = 0\\ \frac{\partial\sigma_{12}}{\partial X_1} + \frac{\partial\sigma_{22}}{\partial X_2} + \frac{\partial\sigma_{23}}{\partial X_3} = 0\\ \frac{\partial\sigma_{13}}{\partial X_1} + \frac{\partial\sigma_{23}}{\partial X_2} + \frac{\partial\sigma_{33}}{\partial X_3} = 0 \end{array} \quad\;\text{in}\;\body_0\,.\tag{3.1.3} \end{equation}
