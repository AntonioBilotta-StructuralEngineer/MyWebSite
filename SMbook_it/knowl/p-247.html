La trasformazione associata è

\begin{align} x_1 \amp = \func{\chi_1}{X_1, X_2, X_3} = X_1\,,\tag{3.1.12}\\ x_2 \amp = \func{\chi_3}{X_1, X_2, X_3} = X_2\,,\tag{3.1.13}\\ x_3 \amp = \func{\chi_3}{X_1, X_2, X_3} = \lambda\,X_3\,,\tag{3.1.14} \end{align}

dove \(\lambda\) è un parametro generico eventualmente da determinare. Il campo di spostamento \(\vec{u}\) ha componenti

\begin{align} u_1 \amp = x_1 - X_1 = 0\,,\tag{3.1.15}\\ u_2 \amp = x_2 - X_2 = 0\,,\tag{3.1.16}\\ u_3 \amp = x_3 - X_3 = (\lambda - 1)X_3\,,\tag{3.1.17} \end{align}

e il gradiente associato è

\begin{equation} \mat{\nabla u} = \left[\begin{array}{ccc} \regulardiff{u_1}{X_1} \amp \regulardiff{u_1}{X_2} \amp \regulardiff{u_1}{X_3}\\ \regulardiff{u_2}{X_1} \amp \regulardiff{u_2}{X_2} \amp \regulardiff{u_2}{X_3}\\ \regulardiff{u_3}{X_1} \amp \regulardiff{u_3}{X_2} \amp \regulardiff{u_3}{X_3} \end{array}\right] = \left[\begin{array}{ccc} 0 \amp 0 \amp 0\\ 0 \amp 0 \amp 0\\ 0 \amp 0 \amp (\lambda - 1) \end{array}\right]\,.\tag{3.1.18} \end{equation}
