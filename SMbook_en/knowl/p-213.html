The adjective “pure” must not be misleading as it only applies to the arrangement of principal directions just described. In fact, it is sufficient to assume a different arrangement for the principal directions, for example obtained with a generic rotation around the axis \(\vec{e}_3 \text{,}\) to have

\begin{equation*} \tens{\sigma} = \sigma \, \vec{s}_1 \otimes \vec{s}_1\,, \end{equation*}

with

\begin{equation*} \matWp{s}{1} = \left[\begin{array}{c} \cos{\alpha}\\\sin{\alpha}\\0 \end{array}\right]\,. \end{equation*}

From which the following matrix form of the given stress tensor

\begin{equation*} \mat{\sigma} = \sigma \left[\begin{array}{ccc} \cos{\alpha}^2 \amp \cos{\alpha}\sin{\alpha} \amp 0 \\ \sin{\alpha}\cos{\alpha} \amp \sin{\alpha}^2 \amp 0 \\ 0 \amp 0 \amp 0 \end{array}\right]\,. \end{equation*}
