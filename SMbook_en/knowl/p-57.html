A rigorous generalization of the equation (1.3.1) would require additional instruments far beyond the introductory purposes of the present exposition. The final result is however given, result which provides the differential \(d\vec{x}\) relative to the current configuration as a function of the differential \(d\vec{X}\) relative to the reference configuration. For this purpose it si useful to write an expanded version of Equation (1.2.1)

\begin{align} x_1 \amp = \func{\chi_1}{X_1, X_2, X_3}\,,\tag{1.3.2}\\ x_2 \amp = \func{\chi_3}{X_1, X_2, X_3}\,,\tag{1.3.3}\\ x_3 \amp = \func{\chi_3}{X_1, X_2, X_3}\,.\tag{1.3.4} \end{align}

The seeken differential can formulated as follows

\begin{gather} dx_1 = \regulardiff{\chi_1}{X_1}\,dX_1 + \regulardiff{\chi_1}{X_2}\,dX_2 + \regulardiff{\chi_1}{X_3}\,dX_3 \,,\tag{1.3.5}\\ dx_2 = \regulardiff{\chi_2}{X_1}\,dX_1 + \regulardiff{\chi_2}{X_2}\,dX_2 + \regulardiff{\chi_2}{X_3}\,dX_3 \,,\tag{1.3.6}\\ dx_3 = \regulardiff{\chi_3}{X_1}\,dX_1 + \regulardiff{\chi_3}{X_2}\,dX_2 + \regulardiff{\chi_3}{X_3}\,dX_3 \,.\tag{1.3.7} \end{gather}

These are a set of equations which can be expressed as multiplication between a matrix and a vector, i.e.

\begin{equation} \left[\begin{array}{c}dx_1\\dx_2\\dx_3\end{array}\right] = \underbrace{\left[\begin{array}{ccc} \regulardiff{\chi_1}{X_1} \amp \regulardiff{\chi_1}{X_2} \amp \regulardiff{\chi_1}{X_3}\\ \regulardiff{\chi_2}{X_1} \amp \regulardiff{\chi_2}{X_2} \amp \regulardiff{\chi_2}{X_3}\\ \regulardiff{\chi_3}{X_1} \amp \regulardiff{\chi_3}{X_2} \amp \regulardiff{\chi_3}{X_3} \end{array}\right]}_{\mat{F}} \left[\begin{array}{c}dX_1\\dX_2\\dX_3\end{array}\right]\,.\tag{1.3.8} \end{equation}
