The following state of stress

\begin{equation*} \tens{\sigma} = \sigma \, \vec{e}_1 \otimes \vec{e}_1\,,\quad \mat{\sigma} = \left[\begin{array}{ccc} \sigma \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{array}\right]\,, \end{equation*}

characterized by principal stresses equal to \(\sigma_1 = \sigma \text{,}\) \(\sigma_2 = 0 \text{,}\) \(\sigma_3 = 0 \) and principal directions identified by the axis \(\vec{e}_1 \) and the plane defined by the other two axes of the triad, it is usually labeled, based on the sign of \(\sigma \text{,}\) state of “pure” traction or “pure” compression.
