In Eq. (3.1.2) the unknowns are the 6 independent components of the stress tensor while the equations available are only 3. Put in these terms the search for the solution would not even be possible. However, given the simplicity of the data of the problem, it is possible to assume a form the stress tensor field and check if all equilibrium equations, over the domain and on the boundary, are satisfied. In particular, let us consider the following form for \(\tens{\sigma} \)

\begin{equation} \mat{\sigma} = \left[\begin{array}{ccc} 0 \amp 0 \amp 0\\ 0 \amp 0 \amp 0\\ 0 \amp 0 \amp \kappa \end{array}\right]\tag{3.1.10} \end{equation}

valid on the whole domain and with \(\kappa \) a generic constant. The satisfaction of Eq. (3.1.3) is easily verified:

\begin{equation*} \begin{array}{l} \frac{\partial\, 0}{\partial\, X_1} + \frac{\partial\, 0}{\partial\, X_2} + \frac{\partial\, 0}{\partial\, X_3} = 0\\ \frac{\partial\, 0}{\partial\, X_1} + \frac{\partial\, 0}{\partial\, X_2} + \frac{\partial\, 0}{\partial\, X_3} = 0\\ \frac{\partial\, 0}{\partial\, X_1} + \frac{\partial\, 0}{\partial\, X_2} + \frac{\partial\, \kappa}{\partial\, X_3} = 0 \end{array} \quad\;\text{in}\;\body_0\,. \end{equation*}

The satisfaction of Eq. (3.1.6) can be also easily verified:

\begin{equation*} \left[\begin{array}{ccc} 0 \amp 0 \amp 0\\ 0 \amp 0 \amp 0\\ 0 \amp 0 \amp \kappa \end{array}\right] \left[\begin{array}{r}n_1\\n_2\\0\end{array}\right] = \left[\begin{array}{r}0\\0\\0\end{array}\right] \,. \end{equation*}

While imposing Eq. (3.1.4), or Eq. (3.1.5), it can be obtained what follows

\begin{equation*} \left[\begin{array}{ccc} 0 \amp 0 \amp 0\\ 0 \amp 0 \amp 0\\ 0 \amp 0 \amp \kappa \end{array}\right] \left[\begin{array}{r}0 \\ 0\\-1\end{array}\right] = \left[\begin{array}{r}0\\0\\-\bar{t}\end{array}\right] \quad \rightarrow \quad \kappa = \bar{t} \,. \end{equation*}
