evaluation of the deformation gradient (nonlinear transformations).

In the case of nonlinear transformations it is not possible to identify a matrix associated with the transformation but, locally, the gradient of the transformation behaves like a linear transformation. Consider thus, for example, the transformation defined as

\begin{align*} x_1 \amp = \func{\chi_1}{X_1, X_2} = X_1^3\,,\\ x_2 \amp = \func{\chi_2}{X_1, X_2} = X_2^3\,. \end{align*}

The calculation of the partial derivatives gives

\begin{align*} \regulardiff{\chi_1}{X_1} = 3\,X_1^2\,,\amp\regulardiff{\chi_1}{X_2} = 0\,,\\ \regulardiff{\chi_2}{X_1} = 0\,,\amp\regulardiff{\chi_2}{X_2} = 3\,X_2^2 \end{align*}

allowing the evaluation of the deformation gradient

\begin{equation*} \func{\mat{F}}{\vec{X}} = \left[\begin{array}{cc} \regulardiff{\chi_1}{X_1} \amp \regulardiff{\chi_1}{X_2} \\ \regulardiff{\chi_2}{X_1} \amp \regulardiff{\chi_2}{X_2} \end{array}\right] = \left[\begin{array}{cc} 3\,X_1^2 \amp 0 \\ 0 \amp 3\,X_2^2 \end{array}\right] \end{equation*}
