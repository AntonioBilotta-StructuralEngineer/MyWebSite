In order to discuss the state of “pure” shear it is necessary to leave the point of view given by the main directions and speak directly of a stress state characterized by the following matrix form

\begin{equation*} \mat{\sigma} = \left[\begin{array}{ccc} 0 \amp \tau \amp 0 \\ \tau \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{array}\right]\,. \end{equation*}

In a such case and always with respect to the given reference triad \(\vec{e}_1\text{,}\) \(\vec{e}_2\text{,}\) \(\vec{e}_3\text{,}\) it is possible to talk of a state of “pure” shear because the only non zero component of the tensor is \(\sigma_{12}=\sigma_{21}=\tau\text{.}\)
